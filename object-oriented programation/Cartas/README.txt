Criar um programa que simule um jogo de cartas, utilizando classes para representar os
diferentes tipos de cartas, como, por exemplo, cartas de baralho com nipes e valores, ou
cartas de jogos como Uno ou Pokémon, com os seguintes requisitos:
• Criar uma classe genérica chamada "Carta", que contém atributos básicos como
valor e naipe. Em seguida, criar classes específicas para cada tipo de carta, como
"CartaBaralho" ou "CartaPokemon", que herdam da classe "Carta" e adicionam
atributos e métodos específicos de cada tipo de carta.
• Criar uma classe chamada "Baralho", que contém uma lista de cartas e métodos
para embaralhar e distribuir as cartas. A classe "Baralho" também deve ter
método para verificar se há cartas suficientes para distribuir.
• Criar uma classe chamada "Jogador", que contém uma lista de cartas que o
jogador possui, e métodos para receber cartas do baralho, jogar cartas e verificar
se o jogador possui cartas suficientes para jogar.
• Adicionar métodos nas classes específicas de carta que sobrescrevem os
métodos da classe genérica "Carta", por exemplo um método "usar" na classe
"CartaPokemon" que ativa o efeito da carta durante o jogo.
• Criar a classe principal para testar as funcionalidades do programa.
